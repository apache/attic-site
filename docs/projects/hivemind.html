Apache HiveMind

Apache HiveMind moved into the Attic in April 2009. Closely linked to Tapestry 4, it lost activity when Tapestry 5 moved to a new IoC container.

The user mailing list, website, downloads and issue tracker all remain open, though the issue tracker is read-only. See the website at http://hivemind.apache.org for more information on HiveMind.

Discussion on the mailing list indicated that Tapestry IoC was the natural successor to HiveMind.

As with any project in the Attic - if you should choose to fork HiveMind outside of Apache, please let us know so we can link to your project.
